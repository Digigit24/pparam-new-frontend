Additional Notes

The codebase is modular and well organised under /assets with separate folders for css, scss, js, images, fonts and third-party vendor libraries. Header and footer are loaded as shared partials, so site-wide updates take only one edit. The project is ready for deployment on any static host (Netlify, Vercel, GitHub Pages or traditional cPanel).
